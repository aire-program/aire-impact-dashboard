# KPI Definitions

## Participation Metrics
- **Total Registrations**: Count of all sign-ups for workshops.
- **Attendance Rate**: `(Total Attended / Total Registrations) * 100`.
- **Unique Participants**: Count of distinct `person_id`s across all events.

## Impact Metrics
- **Confidence Delta**: `Post-Assessment Score - Pre-Assessment Score`. Calculated per person, per workshop.
- **Readiness Score**: Self-reported score (1-5) on "Overall Readiness" to use AI tools.

## Adoption Metrics
- **Adoption Event**: A recorded instance of a user applying a tool in a specific context (e.g., "Using ChatGPT for Syllabus Design").
- **Intensity**: Self-reported depth of usage (1-10).
- **Risk Flag**: Boolean indicator if the usage scenario carries potential policy or ethical risks.
